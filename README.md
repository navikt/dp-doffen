# dp-doffen


## Ansvarsområder

- En app hvor en utvikler kan slå opp nyttige data om dagpenger
- Lytter på hendelser på rapid og lagrer dette i en base hvor man kan søke


## Utvikling

### Komme i gang

Gradle brukes som byggverktøy og er bundlet inn.

```
./gradlew build
```
